# Тестовый кейс для ИИ-жюри Silver Mercury

Кейс из **SM_2025.json**: **«Гриль-сезон и фестиваль индейки Индилайт 2024»** (project_id 5784). В данных есть и презентация в PDF, и видео, и полные описание, стратегия и результаты.

## Содержимое папки

| Файл | Описание |
|------|----------|
| **description.txt** | Описание проекта, контекст, задача, ЦА, инсайт |
| **results.txt** | Результаты и метрики кампании |
| **strategy.txt** | Стратегия, реализация, каналы |
| **case_data.json** | Все поля формы в одном JSON |
| **video_link.txt** | Ссылка на видео (Rutube) |
| **presentation_link.txt** | Ссылки на оригинальную презентацию (PDF) и отчёт по результатам (PDF) из SM_2025 |

## Как тестировать

1. Откройте интерфейс ИИ-жюри (например http://localhost:5175) и перейдите в **«Загрузить кейс»**.
2. Заполните форму, подставив данные из этой папки:
   - **Название проекта:** Гриль-сезон и фестиваль индейки Индилайт 2024
   - **Номинация / Блок / Год:** 982, 50, 2025
   - **Описание:** из `description.txt` или поле `project_info` в `case_data.json`
   - **Результаты:** из `results.txt` или `project_results` в `case_data.json`
   - **Стратегия:** из `strategy.txt` или `project_strategy` в `case_data.json`
3. **Презентация:** скачайте PDF по ссылке из `presentation_link.txt` и приложите в поле для файлов.
4. **Видео:** ссылка в `video_link.txt` (при наличии поля «ссылка на видео» — вставьте её; иначе приложите файл, если интерфейс позволяет).
5. Отправьте форму и откройте результат оценки.

Источник: SM_2025.json, номинация 982 (A01 FMCG Food & Drink), блок 50.
